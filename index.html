Design Considerations

How to do it?

  1. Setup Wordpress on cpanel
  2. Install Theme
  3. Modify Theme
    • Code Reusability - create functions for repeated actions
    • User-friendly - design the theme thinking that the next user does not have the capability to edit a php file
  4. Browser Testing
  5. Mobile Compatibility Testing
  6. Add Content
  7. Optimize for Search Engines

jQuery

jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers. With a combination of versatility and extensibility, jQuery has changed the way that millions of people write JavaScript.

Sample Code



jQuery(document).ready(function(){
	hidethisform(jQuery('#si_contact_form2 .servicedetails'), jQuery('.si_contact_ex_field2_13'));
	jQuery('input:radio[name="si_contact_ex_field8"]').change(function(){
	    if(jQuery(this).is(':checked') && jQuery(this).val() == 'Service'){
	       	showthisform(jQuery('#si_contact_form2 .servicedetails'), jQuery('.si_contact_ex_field2_13'));
	    } else {
	    	hidethisform(jQuery('#si_contact_form2 .servicedetails'), jQuery('.si_contact_ex_field2_13'));
	    }
	});
	if(jQuery('#si_contact_ex_field2_8_1').is(':checked')) {
		showthisform(jQuery('#si_contact_form2 .servicedetails'), jQuery('.si_contact_ex_field2_13'));
	}
});

function hidethisform(_this, _radio1) {
	_this.parent().hide();
	_radio1.hide();
}
function showthisform(_this, _radio1) {
	_this.parent().show();
	_radio1.show();
}
