Creating Structural Profiles

When making structural profiles in Paragon, the following sequence of tasks are being followed

Create the catalogue administrative elements

  • Make a CATA Element
  • Create 2 Structural Section (STSE) Elements
    • For Profile Elements
    • For Reference Data
  • Make a structural category (STCA) for the reference set storge areas
      • This will contain the following reference sets:
    • Profile Geometry Set (GMSS)
    • Pline Set (PTSS)
    • Data Set (DTSE)
Set the reference data storage area

....For Reference Data


This tells PDMS to store the reference data to the correct storage area when the STCA element of profiles (SPRF) is created.
Create the Profile Category and Name the Reference Elements

  • Create the Profile Category (STCA)
      This will create the following elements:
    • SPRF (owned by the profile STCA)
    • TEXT (owned by the profile STCA)
    • GMSS
    • PTSS
    • DTSE
  • Name the Reference Elements
    Rename GMSS, PTSS and DTSE

Structural Section Fittings

Overview of Section Fittings


    Section Fittings - catalogue components that are referenced via the Spref attribute, from a FITT element for section and walls

Examples of panel fittings

  • Sections - web stiffeners, lifting lugs, insulation protection, baseplates, etc
  • Walls - windows, doors, radiators, electrical switches, fire protection, etc.
